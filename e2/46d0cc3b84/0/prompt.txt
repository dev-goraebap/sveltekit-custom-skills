### `references/build.md`에 추가할 섹션

**공통 컴포넌트 패턴** — `preview/components.js` 파일 하나에 공통 UI 정의:

```js
// components.js
function renderNavbar(title, backHref) {
  return `<nav style="...">
    ${backHref ? `<a href="${backHref}">←</a>` : ''}
    <span>${title}</span>
  </nav>`;
}

function renderSidebar(activeMenu) {
  const menus = [
    { id: 'home', label: '홈', href: '02-dashboard.html' },
    { id: 'list', label: '목록', href: '03-list.html' },
  ];
  return `<aside style="...">
    ${menus.map(m => `<a href="${m.href}" class="${m.id === activeMenu ? 'active' : ''}">${m.label}</a>`).join('')}
  </aside>`;
}
```

각 HTML에서 불러오는 방법:
```html
<script src="./components.js"></script>
<script>
  document.getElementById('navbar').innerHTML = renderNavbar('목록', 'index.html');
  document.getElementById('sidebar').innerHTML = renderSidebar('list');
</script>
```

- `file://` 프로토콜에서 동작함 (외부 `fetch` 없이 `<script src>` 방식이므로)
- 공통 스타일도 `components.js` 안에서 `<style>` 태그를 `document.head`에 주입해 관리
