# 펫시터 매칭 플랫폼 기획 메모

## 배경
반려동물 보호자가 외출/여행 시 믿을 수 있는 펫시터를 찾기 어려움.
기존 카페/커뮤니티 매칭은 신뢰도와 편의성이 낮음.

## 주요 기능 아이디어

### 사용자 관리
- 보호자 회원가입/로그인 (소셜 로그인 포함)
- 펫시터 회원가입 (자격증, 경력 입력)
- 프로필 관리 (반려동물 정보 등록)

### 매칭
- 지역/날짜/반려동물 종류로 펫시터 검색
- 펫시터 프로필 상세 (후기, 별점, 자격)
- 돌봄 요청서 작성 및 전송
- 펫시터가 요청 수락/거절

### 돌봄 진행
- 실시간 돌봄 일지 (사진 + 메모)
- 보호자-펫시터 채팅
- GPS 기반 산책 기록

### 결제/정산
- 돌봄 완료 후 결제
- 펫시터 정산 관리
- 환불 처리

### 리뷰
- 돌봄 완료 후 별점 + 텍스트 리뷰
- 리뷰 신고

## 참고
- 1차 MVP는 웹(모바일 반응형)으로
- 보호자/펫시터 역할 분리 필요
- 관리자 기능은 2차에
